### 📲 Installing

To install the package, clone and build the repo, then use `npm` to install dependencies.

```bash
git clone git@bitbucket.org:shadow-ui.git
cd shadow-ui
npm i
```

## 🛥 Deployment

To push changes to the theme, update the package version and publish to NPM:

```bash
git version patch
npm publish
```
